# Uma Meia na Tenda 🧦⛺

Repositório de jogos, livros e ferramentas digitais para escuteiros do Corpo Nacional de Escutas.

## Sobre o Projeto

Este site reúne recursos úteis para dirigentes e escuteiros:

- **🎮 Jogos e Atividades**: Jogos para todas as secções
- **📚 Biblioteca**: Livros e recursos bibliográficos essenciais
- **🔧 Ferramentas**: Apps e sites úteis para orientação, trilhos, cifras e mais

## Como Usar

### Ver o Site Localmente

```bash
hugo.exe server -D
```

Abre o browser em: `http://localhost:1313/uma-meia-na-tenda/`

Antes de fazer merge, faz preview da branch localmente e confirma especialmente:
- homepage;
- páginas de detalhe;
- botões de partilha e impressão;
- recursos novos ou movidos.

### Construir para Produção

```bash
hugo.exe --cleanDestinationDir
```

## Contribuir

**Todos são bem-vindos a contribuir!** 

Se quiseres adicionar jogos, livros ou ferramentas ao site:

1. Consulta o [AI_GUIDELINES.md](./AI_GUIDELINES.md) para conhecer os padrões do projeto
2. Consulta o [CONTENT_WORKFLOW.md](./CONTENT_WORKFLOW.md) para trazer recursos espalhados para o repositório sem criar mais duplicação
3. Trabalha numa branch própria
4. Cria as tuas alterações seguindo as guidelines
5. Abre um Pull Request

Para dúvidas ou sugestões, abre uma issue no repositório.

**Bom Escutismo!** 🏕️

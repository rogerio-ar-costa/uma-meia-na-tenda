O Gerador de Programas é uma ferramenta online que ajuda dirigentes a planear e organizar atividades escutistas.

## Funcionalidades

* Criação rápida de programas de reunião
* Biblioteca de atividades pré-definidas
* Gestão de tempo e recursos
* Exportação para PDF
* Partilha com a equipa de animação

## Como Usar

1. Acede à ferramenta através do link abaixo
2. Cria uma conta gratuita
3. Seleciona o tipo de atividade (reunião, acampamento, etc.)
4. Adiciona atividades da biblioteca ou cria as tuas próprias
5. Define durações e recursos necessários
6. Exporta o programa final

## Link para a Ferramenta

[Aceder ao Gerador de Programas](https://exemplo.com/gerador)

## Outras Ferramentas Úteis

* Calculadora de materiais para acampamento
* Gestor de presenças
* Base de dados de jogos
* Plataforma de comunicação com pais

## Dicas

* Planeia sempre atividades alternativas para mau tempo
* Envolve os jovens no planeamento
* Equilibra atividades físicas e calmas
* Não esqueças os tempos de reflexão
